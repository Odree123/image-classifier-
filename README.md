This project identifies whether the uploaded image is a fruit or vegetable.it uses Convolutional Neural Network (CNN) on a custom dataset deployed using  Streamlit for easy web interaction.
Setting up the environment
# 1.Install python
# 2.create a virtual environment
# 3. python3 -m venv venv
# 4.Activate the virtual environment
#source venv/bin/activate
install pacakages inside the virtual environment:
#pip install numpy tensorflow pillow matplotlib opencv-python streamlit
Run your scripts:
#python train_model.py
my_module.py will be created in your folder
Run your streamlit app
#streamlit run app.py
